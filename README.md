# X Clone

This is an end-to-end clone of X (formerly, Twitter) made using Next.js

## Tech Stack
1. Next.js
2. Redux toolkit

## Getting Started

### Installation
1. git clone https://github.com/starDust1703/X-Clone.git<br>
2. cd X-Clone<br>
3. npm install

### Running the Dev Server
npm run dev

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
